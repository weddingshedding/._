# Wedding Shedding - Simple Static Website

This is a simple static website for GitHub Pages. It does not use PHP, MySQL or admin panel.

## How to upload on GitHub

1. Open your GitHub repository.
2. Delete old PHP files if you want only this simple version.
3. Upload all files from this folder directly to the repository root.
4. Make sure `index.html` is visible in the root.
5. Go to Settings > Pages.
6. Select branch: `main`, folder: `/root`.
7. Save and wait 1-2 minutes.

## Important

GitHub Pages will work only with HTML, CSS and JavaScript. Admin upload features need PHP hosting, so they are not included here.
