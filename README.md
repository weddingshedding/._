# Wedding Shedding - Simple GitHub Pages Website

This is a static HTML/CSS/JS website. It works on GitHub Pages without PHP, MySQL or admin panel.

## Upload to GitHub

1. Unzip this project.
2. Upload all files to the root of your GitHub repository.
3. Make sure `index.html` is directly in root.
4. Go to GitHub repository Settings > Pages.
5. Select branch `main` and folder `/root`.
6. Save and wait 2-5 minutes.
